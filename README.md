## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ and npm
- (Optional) Ollama for local AI models

### Installation

```bash
# Clone the repository
git clone https://github.com/travisbrown348/synthetix-ai-network.git
cd synthetix-ai-network

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Ollama Setup (Optional)
```bash
# Install Ollama
curl -fsSL https://ollama.ai/install.sh | sh

# Start Ollama service
ollama serve

# Pull a model (example)
ollama pull llama2
```
